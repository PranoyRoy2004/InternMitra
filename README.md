## 🧠 Overview

**InternMitra** is an AI-powered career intelligence platform that helps students **discover internships, optimize resumes, and receive personalized career guidance** through an interactive conversational interface.

Unlike traditional job portals, InternMitra focuses on **decision-making assistance**, leveraging AI to bridge the gap between **student potential and industry expectations**.

---

## ✨ Core Features

* 🔍 **Context-Aware Internship Discovery**
  Matches opportunities based on user intent, skills, and career goals

* 🤖 **AI Career Assistant (Chat-based UX)**
  Provides real-time guidance on internships, resumes, and interview prep

* 📄 **Resume Intelligence System**
  Analyzes resumes and generates actionable improvement insights

* ⚡ **Low-Latency API Architecture**
  Fast response times using optimized backend endpoints

* 🎯 **User-Centric Design**
  Minimal, responsive UI focused on usability and clarity

---

## 🏗️ System Architecture

```
Client (HTML/CSS/JS)
        ↓
Frontend UI Layer
        ↓
REST API (FastAPI)
        ↓
AI Processing Layer
        ↓
External APIs / Logic Engine
```

### Key Design Decisions:

* **Separation of concerns** → clean frontend/backend split
* **Stateless API design** → scalable and easy to deploy
* **Modular backend structure** → easy to extend with new AI features
* **Async request handling** → improved performance under load

## 🎯 Problem Statement

Students face three major challenges:

1. Lack of **relevant internship discovery**
2. Poor **resume quality and feedback**
3. No **personalized career guidance**

**InternMitra solves this by combining AI + structured workflows**, enabling smarter career decisions.

## 🚀 Getting Started

```bash
# Clone repository
git clone https://github.com/PranoyRoy2004/InternMitra.git

# Navigate to project
cd InternMitra

# Create virtual environment
python -m venv venv
venv\Scripts\activate

# Install dependencies
pip install -r requirements.txt

# Run server
uvicorn app:app --reload
```
